# Telegram File Bot

This is a project created to strengthen my Python skills using the Telebot library.

## Project Description

The project is a Telegram bot that allows users to:
- save files
- delete files
- search and retrieve files

The bot also communicates with a backend API built using the FastAPI framework, which handles data storage and database operations.

## Technologies Used

- Python
- Telebot (pyTelegramBotAPI)
- FastAPI
- Database (SQLAlchemy)
- Linux (planned deployment)

## Future Plans

- Deploy the project on a Linux server
- Improve database structure and performance
- Expand API functionality

## Goal of the Project

The main goal is to practice Python development, backend architecture, and working with APIs and databases in a real-world style project.
